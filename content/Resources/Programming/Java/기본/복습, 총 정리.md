## 클래스와 데이터
- 클래스는 원시 데이터 유형을 통해 정의할 수 있는 사용자 정의 데이터 유형이다.
- 객체를 생성하면 자바는 메모리 어딘가에 이 객체에 접근할 수 있는 참조값(메모리 주소)를 반환한다.
- 즉, `Class a = new Class();` 코드에서, `a` 변수가 객체에 접근할 수 있는 메모리 주소를 가지게 된다.
