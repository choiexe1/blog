## 타이틀 설정
타이틀 설정은 기본적으로 \`\`\`java title:타이틀 같은 형태로 작성한다.

띄어쓰기가 포함된 긴 타이틀의 경우 title 인자를 "" 따옴표로 감싸 작성한다.

```java title:"ABC DEF"
```

## 폴딩

`fold` 옵션을 사용해 코드블럭이 접혀있는 상태를 나타낼 수 있다.
