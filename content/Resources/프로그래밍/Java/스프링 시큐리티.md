## 스프링 시큐리티
스프링 시큐리티는 사용자 정의 가능한 인증 및 접근 제어 프레임워크이다. 이는 Spring 기반 애플리케이션 보안을 위한 사실상의 표준이다. 

스프링 시큐리티는 Java 애플리케이션에 인증과 권한 부여를 모두 제공하는 데 중점을 둔 프레임워크이다.

스프링 시큐리티를 사용하면 웹 어플리케이션 전반, 인증 및 접근 제어 기능을 간단하게 구현할 수 있다.
