```mermaid
graph LR

Controller --> Service --> Repository --> DB
```

**프레젠테이션 계층**
- UI와 관련된 처리 담당
- 웹 요청과 응답
- 사용자 요청 검증
- 주로 서블릿과 HTTP 같은 웹 기술, 스프링 MVC

- 비즈니스 로직 담당
