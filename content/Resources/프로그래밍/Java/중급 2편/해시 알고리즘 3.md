## 직접 구현하는 Set 3 (직접 만든 객체 보관)
`MyHashSetV2`는 `Object`를 받을 수 있다. 따라서 우리가 직접 만든 `Member`와 같은 객체도 보관할 수 있다.
여기서 주의할 점은 직접 만든 객체가 `hashCode()`, `equals()`를 **반드시** 구현해야 한다는 점이다.
