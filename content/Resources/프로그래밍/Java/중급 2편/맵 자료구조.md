## 컬렉션 프레임워크 - Map 소개 


![[map-data-structure.png]]
맵(Map) 자료구조는 키-값의 쌍을 저장하는 자료구조이다.

- 키는 맵 내에서 유일해야 한다. 그리고 키를 통해 값을 빠르게 검색할 수 있다.
- 키는 중복될 수 없지만, 값은 중복될 수 있ㄷ
