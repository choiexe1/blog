## 서블릿 필터 - 소개
필터는 서블릿이 제공하는 기술이다. 앞서 [[7. 로그인 처리 1 - 쿠키와 세션|로그인 처리 1]]의 요구 사항을 다시 살펴보자.

```
- 홈 화면(로그인 전)
	- 회원 가입
	- 로그인
- 홈 화면(로그인 후)
	- 본인 이름(누구님 환영합니다.)
	- 상품 관리
	- 로그 아웃
- 보안 요구사항
	- 로그인 사용자만 상품에 접근하고, 관리할 수 있음
	- 로그인 하지 않은 사용자가 상품 관리에 접근하면 로그인 화면으로 이동
- 회원 가입, 상품 관리
```

**공통 관심사항**

요구사항을 보면 로그인 한 사용자만 상품 관리 페이지에 들어갈 수 있어야 한다. 앞에서 로그인을 하지 않은 사용자에게는 상품 관리 벝
