## 웹 어플리케이션과 싱글톤
- 스프링은 태생이 기업용 온라인 서비스 기술을 지원하기 위해 탄생했다.
- 대부분의 스프링 어플리케이션은 웹 어플리케이션이다. 물론 웹이 아닌 어플리케이션 개발도 얼마든지 할 수 있다.
- 웹 어플리케이션은 보통 여러 고객이 동시에 요청을 한다.
