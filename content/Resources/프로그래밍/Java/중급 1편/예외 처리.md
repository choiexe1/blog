## 예외 처리가 필요한 이유
외부 서버와 통신할 때는 다음과 같은 다양한 문제들이 발생한다.

- 외부 서버와 연결에 실패한다. (네트워크 오류 등등)
- 데이터 전송에 문제가 발생한다.

강의에서는 실제 네트워크 통신이 아니라 예외가 필요한 상황을 만들기 위해서 네트워크 통신 예제를 통해 왜 필요한지 알려준다.
