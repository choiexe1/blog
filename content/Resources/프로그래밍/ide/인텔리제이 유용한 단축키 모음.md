#### psvm

클래스 바디에서 사용한다.

`public static void main(String[] args) {}`, 진입점인 메인 함수를 생성한다.

#### sout
메서드 바디에서 사용한다. `System.out.println();`을 자동으로 생성한다.

#### soutv

입력이 끝난 표현식을 출력하고 싶을 때 사용한다.  점(.)을 먼저 찍고 입력하면 해당 표현식을 `System.out.println()` 함수에 입력해준다.
